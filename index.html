DAWN: Dual Augmented Memory Network for Unsupervised Video Object Tracking

Project Page

Overview

This webpage contains visual demostration for paper DAWN: Dual Augmented Memory Network for Unsupervised Video Object Tracking. There will be a comparison between DAWN and other state-of-art trackers, especially, C-COT the most precise tracker of VOT 2016 and 2017, and DaSiam the champion of VOT 2018 real-time competition.

Comparison with other trackers

Compared with other state-of-art trackers, DAWN is robust against partial and total occlusion (OC), severe motion blur (MB), abrupt changes in target appearance (AC), multiple object instances, and similar foreground and background features (SB).
